# Hand-Gesture-Calculator-Control-with-Hands-and-Voice
Gesture-based calculator using MediaPipe for hand tracking and speech recognition for voice commands. Detects finger counts for numbers and applies arithmetic operations through spoken input. Provides on-screen results and audio feedback for a hands-free, accessible interaction.
